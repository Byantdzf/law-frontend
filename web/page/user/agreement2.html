委托服务协议

（2019）虎甲委托协议第0001号

本协议由接受服务的一方（以下简称“委托方”）与接受委托的一方（以下简称“受托方”）就本协议项下约定的委托事项达成的一致意见。本协议依据《中华人民共和国合同法》和《中华人民共和国网络安全法》及其他有关法律法规规定而制定。

1、委托方是指通过“虎甲律师问答”网络平台聘请专业律师为其提供法律服务的人，包括自然人和法人。受托方是指虎甲网络科技（深圳）有限公司，接受委托方委托通过“虎甲律师问答”网络平台完成委托事项。

1.1 委托方若非中华人民共和国公民或企业法人，委托事项限于中华人民共和国境内。

1.2 专业律师仅限于经中华人民共和国司法部核准执业的现行注册律师。

2、委托事项

2.1 委托方委托受托方完成其选定的具体委托事项，委托方就其选定的具体委托事项完成付费后，视为对受托方的委托，也视为受托方接受委托。

2.2 委托事项的主要内容是委托方委托受托方就其选定的具体委托事项通过“虎甲律师问答”网络平台选定专业律师提供相关法律服务。

2.3 委托方就其选定的委托事项确保正当合法的委托动机与用途，受托方仅限于就委托方付费的具体委托事项提供相关委托服务。就委托方的委托动机和委托事项用途不具有审查义务，就该委托动机和委托事项用途的非法性及其产生的后果不承担任何法律责任。

3、委托

3.1 委托方选定具体的委托事项应当按照要求简明扼要描述委托要求，认真阅读本委托服务协议，然后支付委托费用。

3.2 委托方支付费用后视为接受本委托服务协议，并委托受托方在“虎甲律师问答”网络平台系统为其选定的委托事项选定专业律师并为其提供相关法律服务。

3.3 委托方同意受托方根据其选定的具体委托事项指定专业律师为其提供法律服务，同意在不能指定律师或委托方指定的律师在网络平台系统规定的时间内没有接受指定的条件下由“虎甲律师问答”网络平台系统自动选定专业律师为其提供法律服务。

3.4 委托方就其选定的委托事项授权受托方并以受托方名义与专业律师另行签订《委托合同》，注明具体的委托事项、律师费用、完成法律服务的期限并按照专业律师的要求提供相关资料。

3.5 受托方就委托方委托事项选定专业律师后，除咨询事项外受托方告知或通过网络平台显示方式提供给委托方专业律师的联系方式，委托方应当就委托事项及时与专业律师进行联系并确定具体委托事项的要求、完成期限、配合提交有关资料等完成事项。

4、委托费用

4.1 委托方就选定的委托事项按照“虎甲律师问答”网络平台规定向受托方支付委托费用，该费用包括支付给受托方的委托服务费用和委托方委托受托方代为支付给专业律师的法律服务费用。

4.2 委托方同意其支付的委托费用中咨询类百分之五十用于支付受托方委托服务费用，另外百分之五十用于支付专业律师咨询费；其他委托事项百分之四十用于支付受托方委托服务费用，百分之六十由受托方代委托方支付给专业律师作为法律服务费用；全案委托根据实际签订的内容予以确定。委托方同意并授权受托方根据市场行情予以适当调整上述费用比例。

4.3 就委托方选定的委托事项除委托方按照本协议支付的委托费用外，受托方不再收取任何费用，委托方也无需另行向专业律师支付任何服务费用。委托方指定的专业律师以相关名义向委托方增加服务费用的，由委托方自行决定，受托方就此不承担任何责任。

4.4 委托方选定了具体委托事项并支付委托费用后又决定取消委托时按照如下内容处理委托方已经支付的委托费用：

4.4.1 “虎甲律师问答”网络平台就委托事项没有选定专业律师之前，委托方可以申请退还委托费用中的法律服务费用；受托方收取的服务费用不予退还。

4.4.2 “虎甲律师问答”网络平台就委托事项已经选定专业律师之后，委托方不得要求受托方收取的委托费用，但是委托方有权要求受托方为其更换专业律师。

4.4.3 委托事项选定专业律师的标志以专业律师通过“虎甲律师问答”网络平台登记或显示已经接受为委托方选定的委托事项提供法律服务。

5、发票出具

5.1 委托方所支付的委托费用，在委托事项全部完成后可以要求开具发票。委托方同意咨询类费用累计人民币壹仟元时由受托方开具发票，受托方就委托方上述金额以下的咨询不再另行单独开具发票；其他委托事项委托方可以按当次委托费用要求受托方出具发票。

5.2 受托方收取的服务费用由受托方按照实际收取的金额向委托方出具发票，受托方代委托方收取并实际支付给专业律师的法律服务费用，由专业律师向委托方开具发票，委托方在委托事项完成后应当向专业律师索取发票。受托方就委托方上述委托受托方代收代付的法律服务费用不承担发票出具以为。

5.3 委托方收到发票后发生退费的，受托方按照国家规定的增值税税率扣除税点，余款在受托方确定退费后的24小时退还至委托方提供的收款账户。

5.4 受托方有权按照国家规定采取电子发票的方式向委托方出具发票，就法律服务费用发票出具具有协助委托方取得该发票的义务。

6、指定律师

6.1 委托方同意受托方根据委托方填写的简要说明为委托事项在“虎甲律师问答”网络平台选定专业律师完成委托方选定的具体委托服务事项，受托方不能人工选定时由“虎甲律师问答”网络平台按照系统推送规则自动为委托方推送专业律师。

6.2 受托方就“虎甲律师问答”网络平台内登记注册的专业律师具有执业资格审查义务，确保为委托事项提供法律服务所选定或系统自动推送的专业律师具有合法有效的律师执业资格。

6.3 委托方接受受托方选定或系统自动推送的专业律师后，应当根据受托方提供的专业律师联系方式及时与专业律师取得联系，就选定的具体委托事项与专业律师进行沟通并确定相关事宜，包括向专业律师详细说明完成委托事项的具体要求与完成期限、交付方式及发票索取等具体委托内容。

6.4 除非受托方选定或系统自动推送的专业律师不能完成委托事项外，委托方在其选定的具体委托事项完成前不能就受托方选定或系统自动推送的专业律师要求受托方进行更换和终止委托。经受托方同意，委托方可以要求更换服务律师；委托方终止委托的，受托方就委托事项所收取的委托费用全部不予退还。

6.5 受托方选定或系统自动推送的专业律师在完成委托事项过程中出现违规或损害委托方委托利益情形，委托方可以向受托方进行投诉。受托方接到投诉后，应当及时就有关情形进行调查并告知委托方调查处理结果。

6.6 受托方应当维护委托方合法权益，认真合法地为委托方选定的委托事项提供委托服务。委托方在委托事项完成过程中应当维护受托方合法权益，不得串通损害受托方利益；应当听从专业律师就委托事项的合理要求和建议。

7、委托事项完成

7.1 专业律师完成委托事项的主要标志就是向委托方交付法律文书或诉讼仲裁文书，以及足以证明委托事项完成的其他有关资料。委托方收到后应当及时签收并予以认真阅读；需要就相关内容进行修改的，应当及时与专业律师联系。针对修改的内容，委托方无需另行向专业律师支付法律服务费用。

7.2 委托方接受专业律师交付的法律文书或诉讼仲裁文书后，应当通过手机客户端向“虎甲律师问答”网络平台确认委托事项完成。受托方依据委托方上述确认按照确定时间向完成委托事项的专业律师支付法律服务费用，委托方接受专业律师交付的法律文书或诉讼仲裁文书后没有提出修改要求或自接受48小时内没有通过手机客户端向“虎甲律师问答”网络平台确认委托事项完成的，“虎甲律师问答”网络平台将自动确认专业律师已经完成委托人具体的委托事项，并按照规定向专业律师支付法律服务费用。

7.3 委托方向“虎甲律师问答”网络平台确认委托事项完成后就收到的法律文书提出修改要求的，原则上向完成上述委托事项的专业律师提出，免费修改以一次为限。

7.4 专业律师交付的法律文书是指委托方要求专业律师完成委托事项的文件，比如起草的协议合同、见证书、律师函、法律意见书、起诉书、上诉书、申请书、辩护词和代理词等。委托立案的为案件受理通知书，委托办理诉讼保全的为诉讼保全裁定书，委托开庭的为该案判决书、调解书或裁决书、裁定书等，委托调查取证的为具体的证据资料或调查报告。委托方应当仔细阅读上述法律文书和诉讼仲裁文书。

7.5 受托方一旦向专业律师支付法律服务费用后，视为受托方全部完成委托方的委托事项。此后，受托方将不会接受委托方提出的关于委托事项法律服务存在瑕疵的任何异议，也不会接受委托方关于该委托事项退费的任何要求。

8、退费申请

8.1 咨询类委托费用一律不予退费，委托方支付费用后系统按照规则将委托方咨询事项推送给专业律师，直到由专业律师完成咨询服务为止。

8.2 日常法律服务和拆分法律服务按照如下规定退费：

8.2.1 委托方就其选定的委托事项付费下单后5个工作日内没有专业律师接单的，委托方可以在上述5个工作日内申请退费，网络平台审核后按照70%比例退费。

8.2.2 委托方就其选定的委托事项付费下单后5个工作日后没有专业律师接单的，委托方可以在上述5个工作日后申请退费，网络平台审核后按照80%比例退费。

8.2.3 委托方就其选定的委托事项付费下单后30日内没有专业律师接单的，委托方无需申请退费，网络平台审核后按照100%比例自动退费。

8.2.4 受托方就委托方选定的委托事项选定或推送专业律师后或专业律师完成委托事项的，受托方不接受委托方任何退费申请。

8.2.5 委托事项完成之前委托方因为专业律师原因要求终止委托但不接受受托方为其委托事项更换专业律师的，受托方有权就委托事项停止服务，委托方委托费用全部不予退还。

8.3 全案委托有关退费按照受托方与委托方签订的《委托合同》规定。

8.4 退费时由受托方按照委托方委托费用支付路径原路退回给委托方，受托方不接受委托方其他退费收款账号。

9、损害赔偿

9.1 受托方接受委托方委托后，因委托本身引起委托方损害应当依法承担赔偿责任，但是赔偿金额以委托方支付的委托费用为最高限额。

9.2 因受托方选定或网络系统自动推送的专业律师原因造成委托方损失的，受托方不承担任何赔偿或连带赔偿责任。

9.3 除按照本协议支付的服务费用外，委托人向受托方选定或网络系统自动推送的专业律师及其推荐的其他任何第三方支付的任何费用，受托人均不予承担退还或赔偿责任。

9.4 受托方就专业律师交付的法律文书或诉讼仲裁文书在使用中造成委托方损害的，受托方不承担任何赔偿责任。受托方就委托方委托事项的胜败等处理结果所产生的损失不承担任何赔偿责任。

9.5 本条款为受托方免责条款，用黑体字加粗突出显示，委托方应当仔细阅读。委托方支付委托服务费后视为就本条免责条款已经详细阅读且清楚该免责条款的所有含义。

10、投诉处理

10.1 委托方在委托事项完成过程中有问题需要投诉的，首先应当联系“虎甲律师问答”网络平台首页注明的投诉电话或客服并详细说明需要投诉的内容，要求及理由。

10.2 受托方收到委托方投诉意见后应当在24小时内委派专人就投诉事项进行调查，受托方调查后及时将调查结果及处理意见回复给委托方。

10.3 委托方就受托方调查后处理意见不服的，可以按照本协议约定的争议条款遵循法律途径解决。

10.4 除合法途径解决争议外，委托方不得采取电话骚扰威胁、恐吓或散布不正当言论等非法方式。否则，由此给受托方造成的名誉或经济损失，委托方应当承担全部赔偿责任。

11、争议解决

11.1 因本协议发生争议，委托方与受托方应当友好协商解决。

11.2 协商不成的，任何一方均有权将争议提交深圳仲裁委员会裁决。
